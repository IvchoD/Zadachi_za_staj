# 🍑 Баба Стринка

## Описание

„Баба Стринка“ е уеб приложение за продажба на домашно приготвени компоти. За да работи трябва в папката backend да се отвори terminal и да се напише "dotnet run".  
Използвано е също Swagger (http://localhost:5251/swagger/index.html) за проверяване на backend.
## Използвани технологии

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

## Възможности

Потребителят може да:

- Разглежда всички продукти
- Търси продукти
- Разглежда подробна информация за продукт
- Добавя продукти в количката
- Променя количеството в количката
- Премахва продукти от количката
- Завършва поръчка
- Регистрира нов профил
- Влиза в съществуващ профил
- Изпраща съобщение чрез страницата „Контакти“

## Автор

Frontend  е направен от мен.

Backend  е направен с помощта на AI (ChatGPT).
