# Електронната изба на баба Стринка 🫙

Малък сайт за домашна зимнина + вграден AI помощник, на когото можеш да
задаваш въпроси за доставки, поръчки, продукти, цени и плащане.

## Пускане

```bash
python3 ai/server.py
```

После отвори **http://localhost:8000** и цъкни буркана 🫙 долу вдясно.

Нужен е само Python 3 — **нула външни зависимости** (никакво `pip install`).

## Какво е AI-то вътре?

Истински **спайков трансформър** (по модела на Spikformer), написан на чист
Python с приоритет лекота пред производителност:

- **Спайкови неврони (LIF)** — невроните си говорят с бинарни спайкове (0/1)
  във времето, с изтичащ мембранен потенциал и праг на изстрелване.
- **Спайкова self-attention** — Q/K/V са спайкови вектори; атенцията е брой
  съвпадащи спайкове (QKᵀV без softmax), смятана event-driven: работа има
  само когато има спайк, затова всичко е леко (~1 ms на въпрос).
- **Reservoir обучение** — блокът на трансформъра е фиксирана произволна
  проекция; обучава се само линеен readout (averaged perceptron) върху
  броя спайкове + bag-of-trigrams skip връзка. Обучава се за ~0.2 s при
  първото пускане и се кешира в `ai/.model_cache.json` (safe за триене).
- **Пазач за домейна** — IDF-претеглено триграмно сходство: на въпроси
  извън темата ботът честно казва „не те разбрах", вместо да си измисля.

Трансформърът разпознава **намерението** на въпроса (14 намерения: статус
на поръчка, срок на доставка, цени, куриери, плащане, връщане…), а
отговорите се сглобяват от шаблони.

## Важно: няма истински бекенд

Данните за доставките са **симулирани**: статуси, куриери, товарителници и
дати се теглят на случаен принцип. Единствената „магия": един и същ номер
на поръчка винаги получава една и съща история (random-ът се сийдва с
номера), така че изглежда като истинска система.

## Файлове

| Файл | Какво прави |
|---|---|
| `ai/spiking_transformer.py` | Спайковият трансформър + readout + кеш |
| `ai/baba_ai.py` | Обучаващи фрази, намерения, генератори на отговори |
| `ai/server.py` | HTTP сървър: статичните файлове + `POST /api/chat` |
| `chat.js` | Чат джаджата (инжектира се на всички страници) |

## Тестване и надграждане

```bash
python3 ai/baba_ai.py --test            # тест с невиждани въпроси (14/14)
python3 ai/baba_ai.py "къде ми е поръчката"   # единичен въпрос
python3 ai/baba_ai.py                   # интерактивен чат в терминала
```

Ново намерение се добавя в `ai/baba_ai.py`: няколко примерни фрази в
`TRAINING_DATA` + функция-отговор в `ANSWER_BUILDERS`. Кешът се
инвалидира сам и моделът се дообучава при следващото пускане.
