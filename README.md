# 🧙‍♂️ Хранилище на Софтуерните Магьосници (Стажантски Проекти)

Добре дошли в официалното хранилище за стажантските проекти на 10-ти и 11-ти клас! 🚀 
Тук съхраняваме всички "заклинания", "еликсири" и най-вече – **чистия код**, който нашите бъдещи Senior девелъпъри създават по време на своя стаж.

## 📂 Структура на архива

Проектите са разделени в две основни гилдии:

* 🛡️ **`10 клас/` – Фокус върху Front-end магии (HTML, CSS, JavaScript), както и работа с WinForms.
* ⚔️ **`11 клас/` (Старши Магьосници)** – Фокус върху Back-end, бази данни и сложни системни архитектури, както и работа с API.

Вътре във всяка папка ще намерите директория с името на съответния стажант и неговия проект.
